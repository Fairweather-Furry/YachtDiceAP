Tutoroll

  • Roll: Press roll or R key.
  • Lock: Click dice or press number keys.
  • You can roll multiple times.
  • Select categories to score.
  • Press reset (or delete key) and try again.

Yacht Dice - solo play







Logic and gameplay are the same as generating via Archipelago.
The generation method is different however:
1) this uses forwards fill instead of backwards fill,
2) there are no filler items and fewer items early on,
3) there are fewer options.
This might make the offline version feel different.
